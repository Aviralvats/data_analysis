In this project, we analyze the Netflix dataset downloaded from Kaggle to gain insights into movies and TV shows available on Netflix. The analysis focuses on release trends, genres, and content distribution over the years.
Dataset
Source: Kaggle Netflix Dataset
Link: https://www.kaggle.com/datasets/paramvir705/netflix-dataset
The dataset contains information about movies and TV shows such as title, type, release year, genre, and other metadata.
Language:Pandas
Libraries: Matplotlib , Pandas
